## Reuse This Framework
0. install `git npm`
1. clone it.
2. `npm install`
3. replace `secure` in `.travis.yml` with your own
4. customize `source/CNAME`
3. customize `_config.yml`
4. Add a cron job `*/5 * * * * /path/to/keep.sh`
